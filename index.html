My Machine Learning Model - Image Edition 📷

the goal of this machine is to idenitify the different type of lego pieces that are used to be built a set. instructions:

1. bring a leg piece like a block, a figure or anything that is a lego piece to do activity

2. show it to the Camera

3. wait for the result to see if its right!!

4. if it right, try using more to see if it can guess it right!! if it not right keep trying!!
